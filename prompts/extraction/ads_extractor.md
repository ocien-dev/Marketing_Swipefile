# Ads Extractor

Use `prompts/extraction/base_insight_extraction.md` as the output contract.

Focus on extracting paid media and ad creative intelligence:

- hooks;
- scroll-stoppers;
- creative angles;
- claims;
- curiosity gaps;
- UGC patterns;
- image/video brief ideas;
- testing hypotheses;
- audience sophistication;
- offer framing for ads;
- platform-specific warnings.

Prioritize insights that help create:

- 10 hooks;
- short ad scripts;
- visual briefs;
- test matrices;
- traffic manager hypotheses.

Special rules:

- Separate creative angle from copy line when possible.
- Mark unsupported performance claims as `hypothesis`.
- If a segment describes an ad that worked, classify as `case` or `example`.
