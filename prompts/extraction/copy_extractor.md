# Copy Extractor

Use `prompts/extraction/base_insight_extraction.md` as the output contract.

Focus on extracting copywriting intelligence:

- hooks;
- headlines;
- leads;
- promises;
- mechanisms;
- objections;
- belief shifts;
- proof;
- CTAs;
- curiosity loops;
- specificity;
- urgency and scarcity;
- narrative transitions.

Prioritize insights that help these agents:

- copy strategist;
- copywriter de VSLs;
- copywriter de anuncios;
- copywriter de webinarios;
- copywriter de quizzes.

Special rules:

- If the source contains usable copy, classify it as `partial_copy` or `complete_copy`.
- If the source teaches a reusable pattern, classify it as `framework`, `template`, or `principle`.
- If the source warns against a copy mistake, classify it as `warning`.
