# Offer Extractor

Use `prompts/extraction/base_insight_extraction.md` as the output contract.

Focus on extracting offer intelligence:

- promise;
- pricing;
- value stack;
- bonuses;
- guarantees;
- urgency;
- scarcity;
- risk reversal;
- checkout;
- order bump;
- upsell;
- low ticket;
- high ticket;
- positioning;
- offer-market fit.

Prioritize insights that help:

- CMO agent;
- copy strategist;
- head de produto;
- estrategista de low ticket;
- copywriter de VSLs.

Special rules:

- If the segment gives a reusable offer structure, classify as `framework`.
- If it gives a concrete offer example, classify as `example` or `case`.
- If it warns about bad offer design, classify as `warning`.
